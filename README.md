# GitHub User Finder

A modern and responsive **GitHub User Finder** built with **HTML**, **CSS**, and **Vanilla JavaScript**. Search any GitHub username and instantly view public profile information using the GitHub REST API.

## Live Demo

🔗 https://github-user-explore.netlify.app/



## Features

- Search any GitHub user
- Fetch data using the GitHub REST API
- Responsive dark-themed UI
- Loading state
- Empty state
- Error state
- Network error handling
- Input validation
- Professional fallback handling for missing data
- Dynamic profile rendering
- Clickable GitHub profile link
- Clickable personal website/blog link
- Formatted join date using `Intl.DateTimeFormat`

## API

This project uses the official GitHub REST API.

```
https://api.github.com/users/{username}
```

Example:

```
https://api.github.com/users/octocat
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Umair-1100/github_user_finder.git
```

Go to the project directory:

```bash
cd github-user-finder
```

Open `index.html` in your browser.

Or use VS Code Live Server.
